# Link na Bio Profissional

Projeto desenvolvido para apresentar uma bio visual e estratégica com foco em posicionamento profissional, portfólio e presença digital.

A proposta da página é reunir, em uma interface única, os principais pontos da minha marca pessoal:

- LinkedIn
- GitHub
- Projetos em desenvolvimento

## Objetivo

Criar uma landing page no estilo "link na bio" com identidade visual forte, linguagem profissional e visual pensado para destacar evolução, consistência e direção de carreira.

## Tecnologias Utilizadas

- React
- Vite
- CSS
- Lucide React

## Funcionalidades

- Foto de perfil personalizada
- Card com acesso direto ao LinkedIn
- Card com acesso direto ao GitHub
- Card de projeto em construção
- Estilo visual em preto, branco e vermelho
- Layout responsivo para mobile e desktop

## Estrutura do Projeto

```bash
src/

```

## Como Executar o Projeto

1. Instale as dependências:

```bash
npm install
```

2. Inicie o ambiente de desenvolvimento:

```bash
npm run dev
```

3. Acesse no navegador:

```bash
http://localhost:5173
```

## Scripts Disponíveis

```bash
npm run dev
```
Inicia o projeto em modo de desenvolvimento.

```bash
npm run build
```
Gera a versão de produção.

```bash
npm run preview
```
Visualiza localmente a versão final de produção.

```bash
npm run lint
```
Executa a análise de qualidade do código.
